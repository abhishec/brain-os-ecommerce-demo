# Nexus Commerce Demo

A demo e-commerce application containing hardcoded business logic for BrainOS discovery.

## Business Logic Examples

This repository contains TypeScript files with embedded business rules that BrainOS can discover, externalize, and convert to dynamic rules:

### Services
- **Pricing** - Tier-based discounts, volume pricing, VIP thresholds
- **Shipping** - Zone-based rates, free shipping thresholds, weight surcharges
- **Eligibility** - Age verification, country restrictions, credit score checks
- **Discounts** - Promo codes, loyalty tiers, seasonal promotions
- **Scoring** - Risk assessment, credit scoring, fraud detection
- **Approvals** - Multi-level approval workflows based on order value

### Hooks
- **useDiscountEngine** - Dynamic discount calculations with promo codes
- **useRiskAssessment** - Real-time transaction risk assessment

## Usage

This is a demo repository for testing BrainOS's ability to scan and discover hardcoded business logic in real codebases.
